Vocab

  • The Internet

    The internet is a network of computers - clients and servers - all interconnected via fiberoptic cables, wires, and wireless connections. Access to the internet is managed by internet service providers like Verizon or Optimum.

  • HTTP

    HHTP stands for HyperText Transfer Protocol. This was a set of rules or standards that defined how to access webpages via the internet. HTTPS later came as a more secure version of the protocol.
